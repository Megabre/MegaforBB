TinyMCE (CDN kullanılmadan) bu klasörden sunulur.

Dosyalar Varient projesinden (tinymce-7.6.1) kopyalanmıştır ve kullanıma hazırdır.

Kullanım:
  Admin panel → Konu ve Post Ayarları → "Metin editörü (konu/mesaj)" alanından
  "TinyMCE" seçin. Konu/mesaj/makale sayfalarında TinyMCE editör açılır.

Klasör yapısı:
  tinymce.min.js   = Ana script
  editor_content.css = Editör içerik stilleri
  skins/, themes/, plugins/, langs/ = TinyMCE bileşenleri
